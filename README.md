# Getting Started (Frontend)

**FitSync** Gym Management system, built using the [T3 Stack](https://create.t3.gg/).

## 1. Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation

First, install the necessary packages using npm:

```bash
npm install

```

### 3. Setup Environment Variables

Copy the example environment file to create your own:

```bash
cp .env.example .env

```

*(Note: Since we are using a separate NestJS backend, you will eventually add the Backend API URL here).*

### 4. Run the Development Server

Once the installation is finished, start the project:

```bash
npm run dev

```

### 5. Open the App

Open [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) with your browser to see the result.
